# TradeFlow – MERN Stock Trading Platform

## 🚀 Overview
A full-stack stock trading platform built using the MERN stack.
Implements real-time market data visualization, authentication, portfolio tracking, and order management.

## 🛠 Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Material UI
- JWT Authentication

## ✨ Features
- Secure user authentication
- Portfolio dashboard
- Holdings & positions management
- Order placement system
- REST API architecture
- Responsive UI

## 📂 Project Structure
backend → Express API and database logic  
frontend → Trading interface  
dashboard → Analytics dashboard
